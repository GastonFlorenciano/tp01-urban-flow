# Urban Flow - Control de Tráfico
## Sprint 1

### Objetivo
Analizar y depurar los registros históricos de infracciones por exceso de velocidad del sistema heredado en la localidad de Vaalserberg para garantizar la integridad de los datos en el nuevo sistema.

### Introducción y Contexto
La localidad fronteriza de Vaalserberg (Bélgica) requiere la migración de datos de radares urbanos. Debido a que el sistema anterior presenta inconsistencias, errores de formato y valores faltantes, este sprint se enfoca en el proceso de ETL (Extract, Transform, Load) inicial para la limpieza de datos.
